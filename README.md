## Описание
GUI-приложение для ведения дневника погоды с возможностью:
- Добавления записей (дата, температура, описание, наличие осадков)
- Фильтрации по дате и температуре (выше заданного значения)
- Сохранения и загрузки данных в JSON-файл
- Валидации ввода
Приложение использует:
*   `tkinter`: Для создания GUI.
*   `random`: Для выбора случайных задач.
*   `json`: Для сохранения и загрузки данных.
*   `os`, `datetime`: Для работы с файловой системой и датой/временем.
